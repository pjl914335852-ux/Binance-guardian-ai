# 🎉 Trading Scout v2.3.0 发布

**发布日期:** 2026-03-06

## 🚀 新功能

### 1. 💻 系统监控

从 Telegram 直接查看服务器状态，无需 SSH 登录！

**功能：**
- ⏰ 系统运行时间
- 📊 CPU 使用率（总计、用户、系统、负载）
- 🧠 内存使用（总计、已用、可用、空闲）
- 💾 磁盘使用（总计、已用、可用、使用率）
- 🤖 机器人进程信息（PID、内存占用）

**使用方法：**
- 发送 `/system` 命令
- 或点击主菜单的 "💻 系统监控" 按钮
- 点击 "🔄 刷新" 更新数据

**示例：**
```
💻 系统监控

⏰ 运行时间: 49 days, 13:13

📊 CPU 使用率:
• 总计: 0.7%
• 用户: 0.5%
• 系统: 0.2%
• 负载: 0.37 / 0.16 / 0.11

🧠 内存使用:
• 总计: 3.8Gi
• 已用: 1.3Gi
• 可用: 2.5Gi

💾 磁盘使用:
• 总计: 75G
• 已用: 30G (42%)
• 可用: 42G

🤖 机器人进程:
• PID: 2892331
• 内存: 84.9 MB

💡 系统运行正常
```

### 2. 🌐 语言同步

UI 语言切换现在会同步到每日摘要！

**改进：**
- 点击 🇬🇧 English → 所有功能变英文（包括每日摘要）
- 点击 🇨🇳 中文 → 所有功能变中文（包括每日摘要）
- 无需重启服务，动态生效
- 统一的语言体验

**技术细节：**
- 语言设置保存到 config.json
- 动态重载语言变量
- 控制台日志也跟随语言

### 3. ⏰ 每日摘要时间优化

根据用户活跃时间调整推送时间：

**之前：** 09:00, 14:00, 20:00  
**现在：** 08:00, 12:00, 20:00

**理由：**
- 08:00 - 早晨起床时间（比 09:00 更早）
- 12:00 - 午餐时间（比 14:00 更活跃）
- 20:00 - 晚上下班后（保持不变）
